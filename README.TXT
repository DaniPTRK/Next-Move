Title: Next Move
Authors: Daniel Ion and Eduard-Eugen Niță
Date: february 2021
Purpose: Educational, part of ADFABER's National Greenfoot Contest
How To Start This Project: This project was made in Greenfoot 3.6.1 version. For starting, open the project in Greenfoot application.
User Instructions: To start the game, use Play button from starting screen. W/A/S/D is used for walk, E for opening items from room, H for tutorial.
